# Portofolio Web - Sarnita
Website portofolio sederhana yang menampilkan profil pribadi, keahlian, dan proyek.  
Proyek ini dibuat sebagai bagian dari tugas kuliah sekaligus latihan membuat versi awal portofolio digital saya.

# Tentang Proyek
Portofolio ini memuat:
- Informasi personal
- Media sosial
- Keahlian
- Proyek yang pernah dilakukan
- Interaksi sederhana menggunakan JavaScript

Website dibuat menggunakan:
- *HTML* → struktur halaman
- *CSS* → tampilan visual
- *JavaScript* → interaksi pengguna

# Fitur Utama
✔ Navigasi halaman (Tentang, Keahlian, Proyek, Kontak)  
✔ Tampilan responsif (bergantung styling CSS)  
✔ Tombol interaktif:
- *Ubah tema warna*
- *Sapa pengguna*


# Cara Menjalankan
1. Download / Clone repository ini.
2. Buka file index.html langsung di browser.

Tidak membutuhkan server atau framework tambahan.
